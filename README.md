# jetski-social-media-app

### Installation steps

#### 1. Download code base

`git clone https://github.com/indigotechtutorials/jetski-social-media-app`

`cd jetski-social-media-app`

#### 2. Install gems 

`bundle install`

#### 3. Start server

`jetski server`

#### 4. View app

Open browser and go to localhost:8000

### Other information

#### Seed database

run the following command to seed database with example records

`jetski db seed`
